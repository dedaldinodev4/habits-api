# Node + Fastify + Prisma + 

A very simple and restAPI for you to use Node, Fastify, Prisma and PostgreSQL together.

# Features

-  Use Node for creating backend
-   Use Prisma for ORM 
-   Use PostgreSQL for database

# Installation

-   Clone repo
-   Run `npm install` or `yarn` or `pnpm`
-   Run `npm run dev` or `yarn dev` or  `pnpm run dev` to start development server
